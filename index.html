したがき動画の設計図メーカー

撮る前・編集する前に、構成の設計図（絵コンテ）を一気に作ります。テーマや素材メモを書いて生成すると、導入→本編→オチのカット割り・尺・テロップ案・画面/BGM・サムネ文言が出ます。カードは手直しでき、Markdownで書き出して編集ソフトの横に置けます。

すでに撮った/話した動画の文字起こしを貼ると、それを見やすい構成表（絵コンテ）に逆算します。編集時にどこで区切り・どんなテロップを入れるかの当たりを付けられます。

⚡ 1秒フック強化モード
冒頭に「0〜1秒で離脱させない」専用シーンを作り、掴みを最優先で設計。縦型ショート向け。

Gemini APIキーの設定

このアプリは Google の Gemini（無料枠・クレカ不要） を直接呼び出します。キーはこの端末の localStorage にのみ保存され、外部には送りません（Googleへの通信を除く）。共有PCでは注意。

  1. Google AI Studio を開く
  2. 「Create API key」でキーを発行
  3. 下の欄に貼り付けて保存
