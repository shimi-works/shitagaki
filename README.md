# したがき ― 動画の設計図メーカー

撮る前・編集する前に、動画の**構成の設計図（絵コンテ）**を一気に作る単一HTMLアプリ。
Google Gemini（無料枠・クレカ不要）をブラウザから直接呼び出します。APIキーは端末の localStorage にのみ保存。

## できること

- **メモから設計**：テーマ/素材メモ＋プラットフォーム/尺/トーンを選ぶと、導入→本編→オチのカット割りを生成
- **文字起こしから起こす**：既存動画の文字起こしを貼ると、構成表（絵コンテ）に逆算
- **🗣 ボイロ解説**：雑学/ニュース系のゆっくり/VOICEROID解説向けに、2キャラの掛け合い台本（読み上げにそのまま使えるセリフ全文＋テロップ）を生成。話者の入替・行の追加削除も可。事実の裏取りを促す注意バナー付き
  - **キャラはプリセット選択**（めたん&ずんだもん／きりたん／琴葉姉妹／ゆっくり霊夢&魔理沙 ほか）。押すだけで口調ごとセット。カスタム入力も可
  - **🔥 いま話題を取得**：ネタを5〜10件表示→クリックでテーマ欄に投入。取得元を選べる：
    - **📰 Yahoo（最新）**：Yahoo!ニュース公式RSSを直接取得（配信時刻つき＝本物の鮮度）。CORS回避に外部プロキシ（allorigins）を1つ経由。ジャンルは傾向に応じてエンタメ/科学/IT/主要などを選択
    - **🔎 AI検索**：Gemini の Google 検索連携で X・5ch/なんJ・まとめ含め横断（鮮度は甘め）
    - **🔀 両方**（既定）：両者をマージ・重複除去し、配信時刻＞when推定で新しい順にソート
  - ネタの傾向（🔥バズ・衝撃／😲へぇ雑学／👀ゴシップ・裏側／📰王道ニュース、既定=バズ）で情報源と方針が変わる。🕒バッジでいつの話題か表示。公式取得でも鮮度・正確性は要確認、噂・ゴシップ系は裏取り必須
- 各シーンに **尺・ナレ・テロップ案・画面/カメラ・BGM・編集ヒント**
- **サムネ文言案**（メイン/補助コピー＋見た目の指示）も同時に生成
- **⚡1秒フック強化モード**：冒頭に「0〜1秒で離脱させない」専用シーンを設計（縦型ショートで自動ON）
- **タイムライン帯**（尺比を色分け）と **合計尺 vs 目標** を表示
- カードは全項目その場で手直し可・並べ替え・追加削除、尺変更は即反映
- **Markdownコピー / 印刷(PDF)** で編集ソフトの横に置ける
- 状態は再訪時に自動復元

## 使い方

1. `index.html` をブラウザで開く
2. 初回に表示される設定画面で [Google AI Studio](https://aistudio.google.com/apikey) の無料APIキーを貼り付け
3. モードを選び、内容を入れて「設計図をつくる」

## 技術

- 単一HTMLファイル（HTML/CSS/JS内包・依存なし）
- Gemini `generateContent`（responseSchema でJSON構造化出力・モデル自動フォールバック）
